# Secure File Transfer System

## Overview
A Java-based secure file transfer system that uses AES encryption to protect files during transmission.

## Features
- AES encryption and decryption
- Secure client-server communication
- SQLite database logging
- File transfer tracking
- Data confidentiality using AES encryption
- Secure file transmission between client and server
- Database-based transfer log management

## Technologies Used
- Java
- AES Cryptography
- SQLite
- Java Socket Programming

## Project Structure
- AESU.java
- SecureClient.java
- SecureServer.java
- DatabaseManager.java

## Usage
1. Start the server
2. Connect the client
3. Select a file to transfer
4. Transfer the file securely
